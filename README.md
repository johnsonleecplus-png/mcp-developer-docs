# 🛠️ MCP 开发者文档中心

> Model Context Protocol 服务器完整指南 | 超过100个MCP服务器详解

## 简介

本项目是一个 MCP（Model Context Protocol）服务器的综合性参考文档，收录了 **100+ 个 MCP 服务器**的详细信息，涵盖了从搜索抓取、数据库存储到 AI 模型、前端开发等 **13 个分类**。

每个 MCP 服务器都包含：
- 📖 功能概述与使用场景
- 🏢 所属机构与背景
- ✨ 核心功能列表
- ✅ 优势与局限性
- 💬 社区评价
- 📦 安装命令
- ⭐ GitHub Stars 与评分

## 本地使用

```bash
# 克隆仓库
git clone https://github.com/johnsonleecplus-png/mcp-developer-docs.git

# 直接用浏览器打开
cd mcp-developer-docs
open index.html
# 或者直接用任意 HTTP 服务器
python3 -m http.server 8080
```

## 功能特点

- 🔍 支持按名称、功能、标签搜索
- 🏷️ 按分类筛选 MCP 工具
- 📊 查看详细信息：背景、功能、优劣势、社区评价
- 📦 一键复制安装命令
- 📱 响应式设计，支持移动端浏览

## 技术栈

纯静态页面：HTML + CSS + JavaScript，无需任何构建工具或后端服务。
